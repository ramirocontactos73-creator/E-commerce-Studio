# 🎧 Studio Store - E-commerce de Equipos Audiovisuales

## 📌 Descripción

Aplicación web de comercio electrónico desarrollada para la gestión y venta de equipos audiovisuales. Permite visualizar productos, agregarlos al carrito y administrar el inventario.

---

## 🚀 Tecnologías

### Backend

* Node.js
* Express
* MySQL

### Frontend

* React (Vite)
* Tailwind CSS

---

## ⚙️ Instalación

### Backend

```
cd backend
npm install
node server.js
```

### Frontend

```
cd frontend
npm install
npm run dev
```
